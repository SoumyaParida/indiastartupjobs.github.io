Quality Assurance Engineers (Comics Team) at Pratilipi

Quality Assurance Engineers (QAEs) at Pratilipi Comics test cutting edge applications and products at the user level and code level, both manually and with automated tools. Our QAEs are able to understand software internals, debug problems using log files, and write automated tests with scripting tools and/or user-level automated tools. They work with dev engineers to understand technical implementation and also with Product Management, User Experience, and Customer Support groups to understand customer usage models and develop test plans and suites that approximate real-world environments. QAEs at Pratilipi Comics are excellent at taking-up tasks from start to completion with minimal help.

Responsibilities:

  • Utilize innovative testing technologies to develop product's testing strategy
  • App, Website, API and Hybrid testing and automation
  • Analyze test logs, debug failures, co-ordinate with stakeholders
  • Co-ordinate with users to understand the issues and try to debug them with stakeholders

Basic Qualifications:

  • Experience in manual testing
  • Experience in UI and API automation testing (Selenium/SOAPUI)
  • Experience in API & Mobile testing
  • Strong knowledge of QA methodology and tools

Bonus Points

  • You are eager to take on more responsibility and assist when needed.
