Job Summary

As a Backend developer with strong knowledge of building stable and scalable applications using NodeJs, You will convert product requirements into technical stories, breakdown into tasks, assign resources, provide clarity on tasks and follow up till completion. And oversee and interact with different stakeholders to gather feedback and clarification

Key Responsibilities:

  • Write reusable, testable, and efficient code following best practices (unit testing, source control, continuous integration, automation, design patterns, etc)
  • This engineer will design and develop API’s using NodeJS based framework.
  • Write clean, well designed code to build optimized and scalable systems.
  • The employee should be able to understand and translate high level requirements to detailed system specifications , build and test planning.
  • Should be able to adapt/ change to any framework and customize/ revamp the framework code.
  • Contribute in all phases of the development lifecycle.

Key Requirements:

  • 3+ years of backend software engineering experience.
  • 1+yr experience in building scalable micro services in NodeJS - with focus, on Rest, data structures.
  • Deep knowledge of RDBMS transactionality or/and no sql.
  • Communication skills to abstract complexity when needed.
  • Skilled in both hands-on development and architectural design
  • Solid interpersonal skills with demonstrated ability to produce results.

We assure you

  • A friendly culture where you are encouraged to share ideas and opinions.
  • A journey that will challenge and reward you in multiple ways.
